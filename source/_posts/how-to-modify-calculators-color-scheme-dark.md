## How To Modify Calculator's Color Scheme (Dark)

 Dark Mode is a special screen setting design that changes the color scheme to display dark themes instead of light ones. It can be used to reduce eye strain and make the overall experience more pleasant while using an app.

 The Windows Calculator App is one such application that allows users to switch between light and dark modes according to their preferences. This tutorial explains how to enable Dark Mode in the Windows Calculator App.
